Welcome to Our Wikipedia Mock-up


About This Project

This Wikipedia mock-up is a project created as part of Harvard's CS50W course. The goal of this project is to provide a platform for learning and sharing knowledge on various topics, much like the real Wikipedia. Here, users can create, edit, and browse articles on a wide range of subjects, from programming languages to historical figures, and everything in between.


Features

  • Comprehensive Articles: Our platform hosts detailed articles on various topics, providing users with in-depth knowledge.
  • User Contributions: Anyone can contribute by creating new articles or editing existing ones, fostering a collaborative learning environment.
  • Search Functionality: Easily find articles using our intuitive search bar.
  • Random Article: Explore something new with a click by viewing a random article.

Why This Project?

Inspired by the structure and functionality of Wikipedia, this project serves as an educational tool to understand the workings of a content management system (CMS) and web development. It showcases the application of Django, HTML, CSS, and providing hands-on experience in building and managing web applications.


Get Involved

We encourage you to dive in and explore the articles available. If you have knowledge to share, feel free to contribute by creating new pages or improving existing content. Your contributions help make this a valuable resource for everyone.
